# Time environment

Python-only units for time management integrations.

## Units

| Unit | Description |
|------|-------------|
| **CalendarICS** | Calendar unit |

See `calendar/README.md` for Calendar setup.

## Using in workflows

1. Use **`add_environment`** with `env_id: time` so **CalendarICS** appears in the Units Library.
2. Set **`environment_type`** to **`time`** when the workflow is time-management-focused.
